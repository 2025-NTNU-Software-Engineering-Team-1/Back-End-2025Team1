# AI Module

This package contains the modular AI system for Normal-OJ.

## Structure
- `service.py`: AI Provider calls (Gemini API)
- `prompts.py`: Prompt templates
- `key_manager.py`: API Key management and rate limiting
- `context.py`: Problem context collection
- `conversation.py`: Chat history management
- `vtuber.py`: Vtuber-specific logic (emotions, persona)
- `exceptions.py`: AI-specific exceptions
- `rag/`: RAG (Retrieval-Augmented Generation) module
